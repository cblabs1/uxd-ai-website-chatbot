== Installation ==

= Automatic Installation =

1. Log into your WordPress admin panel
2. Navigate to Plugins → Add New
3. Search for "UXD AI Website Chatbot"
4. Click "Install Now" then "Activate"
5. Go to AI Chatbot in your admin menu to configure

= Manual Installation =

1. Download the plugin zip file
2. Upload to `/wp-content/plugins/uxd-ai-website-chatbot/`
3. Activate through the 'Plugins' menu in WordPress
4. Navigate to AI Chatbot → Settings to configure

= After Installation =

1. **Get API Key**: Sign up with OpenAI, Anthropic, or Google AI
2. **Configure Settings**: Enter your API key and select your preferred model
3. **Customize Design**: Choose colors, position, and welcome message
4. **Train Content**: Sync your website content for better responses
5. **Test & Enable**: Test the chatbot and enable for your visitors

== Frequently Asked Questions ==

= Do I need an API key? =

Yes, you need an API key from your chosen AI provider (OpenAI, Anthropic, or Google). Most providers offer free credits or trials to get started.

= How much does it cost to run? =

Costs depend on your chosen provider and usage volume:
* OpenAI GPT-4o-mini: ~$0.001 per conversation
* Claude 3 Haiku: ~$0.0005 per conversation
* Gemini 1.5 Flash: ~$0.0002 per conversation (free tier available)

= Is it GDPR compliant? =

Yes! The plugin includes comprehensive GDPR compliance features:
* User data export and deletion tools
* Configurable data retention policies (default 30 days)
* Privacy consent forms
* Cookie consent management
* Anonymized data collection options
* Right to be forgotten support

= Can I customize the chatbot appearance? =

Absolutely! You can customize:
* Widget position (4 corners or center)
* Color scheme and theme
* Welcome messages
* Widget size (small, medium, large)
* Custom CSS for advanced styling
* Animation effects
* Show/hide on specific pages

= Does it work on mobile devices? =

Yes, the chatbot is fully responsive and optimized for all devices including smartphones and tablets with touch-friendly controls.

= Can I train it on my own content? =

Yes! The plugin offers multiple training options:
* Automatic synchronization with your posts and pages
* Manual Q&A training data
* Custom post type support
* Real-time content updates
* Import/export training data (CSV/JSON)

= What about rate limiting and spam protection? =

Built-in protection includes:
* Configurable rate limits per user/IP
* Per-minute and per-hour request limits
* IP blocking for repeated abuse
* Message length limits
* Spam pattern detection
* Blocked words filter

= Can I see conversation logs? =

Yes, the admin panel provides:
* Complete conversation history
* User satisfaction ratings

* Analytics dashboard with charts
* Response time tracking

The AI models support multiple languages naturally. The chatbot will respond in the language the user writes in. The plugin interface is translation-ready with support for WordPress localization.

== Privacy Policy ==

This plugin connects to third-party AI services to provide chatbot functionality. Please review the privacy implications:

**Data Sent to AI Providers:**
* User messages and conversation context
* Website content used for training (posts, pages, custom content)
* Basic usage statistics (optional)

**Data Collected Locally:**
* Conversation logs (optional, configurable retention 1-365 days)
* User satisfaction ratings (optional)
* Analytics data (optional)
* Session information (cookies, optional)

**Your Rights Under GDPR:**
* Export your conversation data at any time
* Request deletion of your conversation data
* Control what data is collected via settings
* Set custom data retention policies
* Opt-out of non-essential data collection

**Third-Party Services:**
The plugin may connect to:
* OpenAI API - [Privacy Policy](https://openai.com/privacy/)
* Anthropic Claude API - [Privacy Policy](https://www.anthropic.com/privacy)
* Google AI/Gemini API - [Privacy Policy](https://policies.google.com/privacy)

**Data Storage:**
* Conversations stored in your WordPress database
* Configurable automatic deletion after retention period
* Secure database encryption recommended
* No data shared with plugin developers

**Cookies:**
* Session cookies for chatbot functionality (can be disabled)
* Analytics cookies (optional, can be disabled)
* No third-party tracking cookies
