= How It Works =

1. **Choose Your AI Provider** - Select from OpenAI, Claude, or Gemini
2. **Enter API Key** - Get your API key from your chosen provider
3. **Customize Appearance** - Match your brand colors and positioning
4. **Train on Content** - Let the AI learn from your website content
5. **Go Live** - Enable the chatbot and start helping visitors!

== Installation ==

= Automatic Installation =

1. Log into your WordPress admin panel
2. Go to Plugins ‚Üí Add New
3. Search for "AI Website Chatbot"
4. Click "Install Now" then "Activate"
5. Go to AI Chatbot in your admin menu to configure

= Manual Installation =

1. Download the plugin zip file
2. Upload to `/wp-content/plugins/ai-website-chatbot/`
3. Activate through the 'Plugins' menu in WordPress
4. Go to AI Chatbot ‚Üí Settings to configure

= After Installation =

1. **Get API Key**: Sign up with OpenAI, Anthropic, or Google
2. **Configure Settings**: Enter your API key and customize appearance  
3. **Train Content**: Sync your website content for better responses
4. **Test & Enable**: Test the chatbot and enable for visitors

== Frequently Asked Questions ==

= Do I need an API key? =

Yes, you need an API key from your chosen AI provider (OpenAI, Anthropic, or Google). Most providers offer free tiers or credits to get started.

= How much does it cost to run? =

Costs depend on your chosen provider and usage volume:
* OpenAI: ~$0.002 per conversation
* Claude: ~$0.008 per conversation  
* Gemini: Free tier available

= Is it GDPR compliant? =

Yes! The plugin includes:
* Data export and deletion tools
* Configurable data retention policies
* Privacy consent forms
* Anonymized data collection options

= Can I customize the appearance? =

Absolutely! You can:
* Choose from 4 positioning options
* Customize colors and themes
* Set custom welcome messages
* Add your own CSS styling
* Control when and where it appears

= Does it work on mobile? =

Yes, the chatbot is fully responsive and optimized for mobile devices with touch-friendly controls.

= Can I train it on my content? =

Yes! The plugin automatically:
* Syncs your posts and pages
* Learns from your content
* Provides contextual responses
* Updates when you publish new content

= What about rate limiting and spam? =

Built-in protection includes:
* Configurable rate limits per user
* IP blocking for abuse
* Message length limits
* Spam detection and filtering

= Can I see conversation logs? =

Yes, you can view:
* All conversations in the admin panel
* User satisfaction ratings
* Usage statistics and analytics
* Export conversation data
